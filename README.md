# ❓ QuizMaster - Interactive Quiz Application

A feature-rich quiz application with multiple categories, difficulty levels, timers, and scoring system. Built with vanilla JavaScript and the Open Trivia DB API.

## ✨ Features
- ✅ 20+ categories to choose from
- ✅ 3 difficulty levels (Easy, Medium, Hard)
- ✅ Adjustable question count (5-20)
- ✅ Timed questions with visual countdown
- ✅ Score tracking with time bonuses
- ✅ Progress bar visualization
- ✅ Instant answer feedback
- ✅ Detailed results page
- ✅ High score persistence
- ✅ Share your scores
- ✅ Fully responsive design

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (ES6+)
- Open Trivia DB API
- Font Awesome Icons
- Local Storage API

## 📦 Installation
1. Clone the repository
2. Open `index.html` in your browser
3. Start playing!

## 🎮 How to Play
1. Select a category
2. Choose difficulty level
3. Set number of questions
4. Set time per question
5. Answer before time runs out
6. Get points + time bonus
7. Check your final score!

## 📊 Scoring System
- **Correct Answer:** 100 points
- **Time Bonus:** 10 points per second left
- **Total Score:** Correct answers + time bonuses
